안녕하세요
2조 코딩이모조 ㅇㅇㅇ 입니다.
저희조는 제주 여행 전 참고하기 좋은 여행사이트를 구현해보았습니다.

저희 조는 코로나로 인해 제주도로 떠나는 여행객들이 증가함에 따라
제주도에서 가볼만한 관광지를 제공할 목적으로 기획했습니다.

그래서 저희는 여행 사이트로 유명한 스투비 플래너, 마이리얼트립, 대한민국 구석구석, 트리플 참고했고, 
그렇게 계획한 주 기능은 제주도의 지역별 장소 정보 제공, 다녀온 사람들의 후기 공유, 마이플래너 입니다.

프로젝트 일정
첫주는 어떤 주제의 사이트를 구현할지, DB설계와 UI설계를 어떻게 할지 구상 및 설계를 하였고,
둘째주 부터 MVC2패턴을 통해 각자 맡은 페이지의 기능을 구현한 후 마지막 날 오류 확인 후 작업 완료하였습니다.

해당 기능을 구현하기 위해 사용한 개발도구 및 환경은
제이쿼리, 깃헙, 소스트리, 이클립스, 자바, 자바스크립트, 오라클, sql developer ,... (뭐가더있죠..)

주 기능 설계는 유스케이스 다이어그램을 통해 정리하였습니다.
해당 기능을 불러올 수 있는 데이터베이스 설계는 erd 클라우드를 통해 설계했고,
회원관리를 제외한 기능으로 틀을 잡아 카카오오븐을 통해 UI를 설계했습니다.

<카카오 오븐으로 설계한 페이지>
메인페이지 입니다. - 메인페이지는 제주도의 지도에서 각 지역으로 이동할 수 있는 기능 및 전 카테고리를 요약한 페이지를 설계했습니다.
회원가입 및 로그인 페이지 입니다. - 개인정보를 입력하여 가입할 수 있는 회원가입 페이지와 로그인 기능을 설계한 페이지입니다.
내 정보 수정 페이지입니다. - 전체 정보 수정 및 암호화된 비밀번호 변경과 회원 탈퇴 기능을 설계하였습니다.
여행지 타입 별 추천지 확인 페이지 입니다. - 메뉴바의 확인하고 싶은 카테고리를 누르면 테마 별 추천 장소의 목록을 확인할 수 있는 페이지입니다.
마이플래너 페이지 입니다. - 여행 일정 등록 및 일정 리스트를 작성 할 수 있게 설계한 페이지 입니다.
여행후기 페이지 입니다. - 여행 후기 작성 및 공유, 댓글 기능을 설계한 게시판 페이지 입니다.

시연하겠습니다.
메인 페이지 - 메인페이지 입니다. 헤더영역엔 홈으로 돌아오는 홈페이지 로고, 카테고리 메뉴바, 로그인/회원가입 버튼을 넣었고,
푸터엔 저희 조 로고, 카피라이트, 조원 이름을 넣었습니다.

메인페이지 상단엔 제주도의 주요 지역을 나눈 지도를 넣었고, 지역 별로 클릭하면 해당 지역의 추천 여행지를 카테고리 별로 볼 수 있습니다.
메인페이지 하단엔 카테고리별 대표 추천 여행지를 넣었습니다.

회원가입 - 회원가입 페이지입니다. 아이디와 휴대폰 번호는 PK,UK로 설정하여 중복하여 가입할 수 없습니다.

로그인 - 회원가입을 통해 db에 등록된 계정으로 로그인 할 수 있습니다.
로그인 후 헤더의 로그인/회원가입 영역은 로그인 한 회원의 이름과 내정보, 로그이웃버튼이로 바뀌고
메뉴바에 마이플래너가 추가됩니다.

내 정보 - 로그인 후 내 정보 버튼을 누르면 아이디와 전화번호를 제외한 회원의 정보를 변경할 수 있는 페이지로 연결됩니다.
	변경할 정보를 입력 후 정보수정 버튼을 누르면 수정된 정보가 바로 반영됩니다. 
	하단의 비밀번호 변경 버튼을 누르면 비밀번호 변경 팝업이 생성되어 비밀번호를 변경할 수 있습니다.
	탈퇴 버튼을 누르게 되면 탈퇴의사를 한번 더 묻는 메시지가 나옵니다. 
            이때 탈퇴를 진행하면 해당하는 계정의 데이터 상태값이 N으로 변경되어 탈퇴 처리가 됩니다.

   (데이터를 삭제한건지 status가 N이 된건지 추가해도 좋을거같습니다~!)

   (시현순서 : 정보수정 -> 비밀번호 변경 -> 탈퇴) 

타입 별 추천지 확인 - 메뉴바의 카테고리를 누르면 각 카테고리 메인화면에서 테마별 추천 여행지를 확인할 수 있습니다.
확인을 원하는 장소의 썸네일을 클릭하면 해당 장소의 정보, 대표 사진들과 지도api로 구현한 위치 확인이 가능합니다.

로그인 시 마이플래너 확인 - 여행하고 싶은 일자의 출발일과 도착일을 선택하면 여행 일수를 확인할 수 있고,
하단에선 이틀치의 to-do 리스트를 작성하고 저장하여 추후에도 확인할 수 있습니다. (확인 안되면 대사변경 부탁드립니다!)

다음은 여행후기 기능 구현 입니다.

상단  메뉴바에서 여행후기 카테고리를 클릭 하면 리뷰리스트 서블릿 경로로 연결하여 여행후기 메인페이지로 이동합니다.
메인페이지는 관광지 타입에 따라 메인페이지에서 후기를 4개씩 볼 수 있습니다.

보고싶은 후기의 썸네일을 누르면 리뷰디테일 서블릿 경로를 통해 상세페이지를 확인할 수 있고,
로그인 한 회원은 댓글을 남길 수 있습니다.
(로그인이 안되있을 시 로그인 페이지로 연결되나 상세페이지가 아닌 홈으로 연결됩니다 ㅠㅠ 언급X)
(댓글 수정/삭제 기능 없습니다. 언급X...)

하단 목록버튼을 누르면 다시 메인페이지로 돌아갈 수 있습니다.

메인페이지 상단 글쓰기 버튼을 누르면
원하는 타입 장소의 후기를 남길 수 있습니다.
(로그인이 안되있을 시 댓글과 동일하게 홈으로 연결됩니다.. 언급X)
